# Customization

Open the theme sidebar by clicking the **half-circle icon** in the top bar (or press the theme key).

## What You Can Customize

| Setting | Description |
|---------|-------------|
| [Themes](themes.md) | 15+ built-in color presets or fully custom colors |
| Background image | URL or file upload with opacity control |
| Surface transparency | Blur and transparency for widget cards |
| Border radius | Widget card corner rounding |
| Border width | Widget card border thickness |
| Layout style | Default or flat layout variant |
| [Custom CSS](custom-css.md) | Inject arbitrary CSS |

## Save Scope

Changes can be saved at two scopes:

- **Global** — applies to all profiles
- **Per-profile** — overrides for the active profile only

The save scope selector is at the bottom of the theme sidebar.
