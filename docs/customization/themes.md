# Themes

New Tab Hub ships with 15+ built-in color presets and a full custom color editor.

## Custom Colors

Select **Custom** from the theme picker to edit individual color tokens:

| Token | Description |
|-------|-------------|
| Background | Page background color |
| Surface | Widget card background |
| Surface Alt | Alternate surface (nested elements) |
| Border | Card border color |
| Text | Primary text color |
| Text Muted | Secondary/dimmed text |
| Accent | Highlight and interactive elements |
| Accent Text | Text on accent background |

Colors can be entered as hex (`#1e1e2e`), RGB, or CSS color names.

## Background Image

Set a background image via:

- **URL** — paste any direct image URL
- **File upload** — upload from your local machine (stored in browser storage)

Adjust **opacity** with the slider to blend the image with the background color.

## Layout Style

Toggle between **default** (with card borders and depth) and **flat** (minimal, borderless) layout styles.

!!! tip
    Theme settings saved globally apply across all profiles. Save per-profile to have different looks for Work and Personal dashboards.
