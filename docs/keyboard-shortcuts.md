# Keyboard Shortcuts

New Tab Hub is designed to be operated almost entirely from the keyboard.

Press `?` at any time to show the shortcuts overlay in the dashboard.

## Global

| Key | Action |
|-----|--------|
| `/` or `Ctrl+K` | Focus the search bar |
| `Escape` | Close dialogs, clear search, blur input |
| `?` | Show / hide keyboard shortcuts help |

## Navigation

| Key | Action |
|-----|--------|
| `Arrow keys` or `h / j / k / l` | Move focus between widgets and items |
| `Enter` | Open the focused link |
| `u` | Scroll page up |
| `d` | Scroll page down |

## Pinned Links

| Key | Action |
|-----|--------|
| `1` – `9` | Open the corresponding pinned link directly |

## Search

| Key | Action |
|-----|--------|
| `Enter` | Execute search / navigate to URL |
| `Arrow Up / Down` | Move through search results |
| `Escape` | Clear search and return focus to grid |

## Edit Mode (Grid)

| Key | Action |
|-----|--------|
| `Escape` | Exit edit mode |

---

!!! tip
    The focus key for search can be changed in the [Search widget config](widgets/productivity/search.md) via the `focusKey` option.
