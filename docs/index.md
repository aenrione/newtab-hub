# New Tab Hub

**A keyboard-first new tab dashboard for Chromium browsers.**

Replace your default new tab page with a configurable grid of widgets — search, pinned links, RSS feeds, market tickers, self-hosted service stats, and more. Built entirely with vanilla JavaScript — no frameworks, no build step, no runtime dependencies.

## Why New Tab Hub?

- **Keyboard-first** — search auto-focuses on every new tab; `1`–`9` open pinned links; `h/j/k/l` navigate widgets
- **No cloud, no accounts** — everything runs locally in your browser
- **Private by design** — a two-tier config system keeps personal URLs out of version control
- **50+ widgets** — productivity tools, self-hosted service integrations, social feeds, and more
- **Zero build step** — load unpacked, edit JS files, reload — done
- **15+ themes** — Gruvbox, Catppuccin, Nord, Dracula, and more; fully customisable
